# Guess The Number

## Description
A JavaScript-based number guessing game where the system generates a random number between 1 and 100 and the user tries to guess it within limited attempts. The app gives high/low feedback and tracks previous guesses.

## Concepts Practiced
- DOM selection and manipulation
- Event listeners
- Random number generation
- Conditional logic
- Input validation
- State tracking with variables and arrays
- Dynamic UI updates

## How to Run
Open index.html in your browser.
